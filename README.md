🏥 Hospital Emergency Room Dashboard (Excel)
📊 Project Overview

The Hospital Emergency Room Dashboard is an interactive Excel-based data analytics project designed to analyze emergency department operations and improve hospital decision-making.

The dashboard transforms raw hospital emergency room data into clear visual insights that help administrators monitor patient flow, waiting times, admissions, and department workload.

Using Excel analytics tools such as Pivot Tables, Charts, and Slicers, the dashboard enables quick identification of operational patterns and performance bottlenecks.

🎯 Business Problem

Emergency departments handle a large volume of patients daily. Without structured data analysis, hospitals struggle to monitor key operational metrics such as:

Patient waiting times

Admission vs discharge rates

Department referral load

Patient demographics

Monthly patient trends

This project solves that problem by building an interactive dashboard that converts raw data into actionable insights for hospital management.

📈 Key Performance Indicators (KPIs)

The dashboard tracks several important healthcare metrics:

Total Patients Treated

Average Waiting Time

Admission vs Non-Admission Cases

Patient Satisfaction Score

Department Referrals

Patient Age Group Distribution

Monthly Patient Trends

These KPIs help monitor emergency room efficiency and service quality.

🔍 Key Insights from Analysis

Some important insights derived from the dataset include:

Emergency room visits show clear monthly patterns, indicating peak operational periods.

Specific age groups dominate ER visits, highlighting demographic demand.

Admission rates vary by case type and department referrals.

Waiting times increase during high patient inflow periods, suggesting staffing optimization opportunities.

These insights help hospitals improve resource planning, staffing decisions, and patient experience.
