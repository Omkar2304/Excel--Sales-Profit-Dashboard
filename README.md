
<h1 align="center">📊 Excel Sales & Profit Dashboard</h1>

<p align="center">
An interactive Excel dashboard for analyzing sales performance, profitability trends,
and product category performance using data visualization techniques.
</p>

<hr>

<h2>📊 Project Overview</h2>

<p>
The <b>Sales & Profit Dashboard</b> is a data analytics project built using Microsoft Excel.
The goal of the project is to transform raw sales data into meaningful business insights
through interactive dashboards and visualizations.
</p>

<p>
This dashboard allows users to track sales performance, profit trends, and product category
performance across different time periods.
</p>

<h2>🎯 Business Problem</h2>

<p>
Businesses generate large amounts of sales data, but without proper analysis,
it becomes difficult to understand performance trends and profitability.
</p>

<p>
This dashboard helps answer important business questions such as:
</p>

<ul>
<li>Which products generate the highest sales?</li>
<li>Which categories generate the most profit?</li>
<li>How do sales trends change over time?</li>
<li>Which segments or regions perform best?</li>
</ul>

<hr>
